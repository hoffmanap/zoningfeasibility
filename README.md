This tool examines the relationship between the density and dimensional standards of El Paso's Zoning Ordinance (Appendix B) and off-street parking standards. To develop this app, Appendix B and Chapter 20.14 (Off-Street Parking) were ingested to determine development feasibility.

To use this app, adjust the lot dimensions to the property you want to analyze. Then, select the zoning district and use and adjust the number of units. The tool automatically applies the required yard standards and parking requirements to alert when the development is no longer feasible.
